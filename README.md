# May Flow Rate Test for Creality K1

Test the max flow rate of your K1 / K1 Max using the pre-sliced GCode.

The test based on the work of Stefan Hermann (@CNCKitchen).
The originally generated GCode did not work on the K1 / K1 Max, so I specially modified it, to be printable on the K1 / K1 Max.
Download the GCode and print it. Then follow the great instructions on Stefan's website to evaluate it (https://www.cnckitchen.com/blog/extrusion-system-benchmark-tool-for-fast-prints.
